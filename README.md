# Android 2D Shooter Game

Android 2D Shooter Game is a simple mobile shooter game developed in Java using Android Studio.  
The project demonstrates basic 2D game development concepts, including sprite rendering, player movement, enemies, shooting mechanics, collision detection, health system, and a custom game loop.

## Features

- 2D player movement using an on-screen joystick
- Sprite-based character rendering
- Tile-based game map
- Enemy spawning and enemy movement toward the player
- Shooting/spell mechanic
- Collision detection between player, enemies, and projectiles
- Player health system
- Health bar rendering
- Game over state
- Custom game loop for updating and drawing game objects
- Android Canvas-based rendering

## Technologies Used

- Java
- Android Studio
- Android Canvas
- Gradle
- Object-Oriented Programming
